# Stopwatch App ⏱️

A clean, responsive stopwatch built with HTML, CSS, and JavaScript.

## 🔧 Features
- Start / Stop time tracking
- Reset to 00:00:00
- Lap recording
- Light/Dark mode toggle
- Responsive design

## 🚀 How to Run
1. Clone the repo
2. Open `index.html` in your browser
